# Foreign Fields

When creating entries, you'll often find dropdown fields that reference other entries of the campaign. These are called **foreign fields**. For example, an [item](/entries/items) has a **character** field, which is its creator.

![Foreign field example](img/foreign.png)

Every foreign field is used by searching for an entry's name and needs at least 3 characters types. You can filter for an exact search by typing `!Dave`, for example if you have multiple _Dave_s in your campaign but only one called _Dave_.

## Creating a new entry

Sometimes you want to add a foreign field, but haven't created it yet. You can use the [quick creator](/features/quick-creator), or click on the little **+ (plus)** sign next to the field. This option only appears if you have the permission to create an entry of that type.

Clicking on the **+ (plus)** actually calls the quick creator. Fill out the form and click save. The new entry is created in the background, and the foreign field is filled out to the newly created entry.
