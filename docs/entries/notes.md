# Notes

The catch-all for anything that doesn't belong in the other categories. Notes can be _nested_ (have a parent note) and be tagged, so combining that and [bookmarks](/advanced/bookmarks), you can create your own new section.

* **Lore** -> notes
* **Recipes** -> notes
* **Todos** -> notes
* **Laws** -> notes
* **Plots** -> notes
* **Everything else** -> Kanka

If you need better separation of content, consider a [custom category](/features/campaigns/categories).
